# Utils lib

Simple library for common constants, whether they are data or functions. Install it with


    <repositories>
        <repository>
            <id>central</id>
            <url>https://repo.maven.apache.org/maven2</url>
        </repository>
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>
    <dependencies>
        <dependency>
            <groupId>com.github.Lucas1774</groupId>
            <artifactId>utils</artifactId>
            <version>x.x.x</version>
        </dependency>
    </dependencies>
